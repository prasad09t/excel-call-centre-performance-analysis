## Project Overview
This project analyzes call centre operational data to evaluate agent performance, call trends, revenue, and customer satisfaction using Microsoft Excel.

## Dataset
The dataset includes:
- Call volume and call duration
- Agent assignment
- Customer demographics
- Purchase amount
- Customer satisfaction ratings

## Tools Used
- Microsoft Excel
  - PivotTables & PivotCharts
  - Excel formulas (SUMIFS, COUNTIFS, AVERAGEIFS, IF)
  - Charts, Slicers, and Dashboards

## Key Analysis
- Agent performance comparison
- Call trends by day and time
- Revenue generated through calls
- Customer satisfaction analysis

## Dashboard
An interactive Excel dashboard was created to monitor KPIs and support data-driven decision making.

## Outcome
The analysis helps identify high-performing agents and peak call periods, enabling better operational planning.
